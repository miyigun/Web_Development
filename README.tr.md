# Web_Development

Bu repo, **web geliştirme örneklerini, öğrenme denemelerini ve mini projeleri** içermektedir.  
Amaç, modern, pratik ve ölçeklenebilir web uygulamaları geliştirmeyi öğrenmek ve paylaşmaktır.

---

## Klasörler / Kategoriler

### 1. Experiments-Simple_Web
- Küçük denemeler ve hızlı prototipler  
- HTML / CSS / JavaScript alıştırmaları  
- Basit UI/UX denemeleri ve component fikirleri  

### 2. Math-Educational_Tools
- Matematik odaklı araçlar ve hesaplayıcılar  
- Eğitsel mini uygulamalar ve görselleştirmeler  
- Öğrenme ve pratik yapmayı destekleyen örnekler  

---

## Başlangıç

Projeleri çalıştırmak için:

1. Repoyu klonlayın:
   ```bash
   git clone https://github.com/miyigun/Web_Development.git
   ```

2. Repo klasörüne girin:
   ```bash
   cd Web_Development
   ```

3. Bir alt klasör seçip projeyi çalıştırın:
   - Statik web projelerinde `index.html` dosyasını tarayıcıda doğrudan açabilirsiniz.
   - Veya local server kullanmanız önerilir.

   Örnek (VS Code Live Server):
   - Klasörü VS Code ile açın
   - `index.html` → sağ tık → **Open with Live Server**
