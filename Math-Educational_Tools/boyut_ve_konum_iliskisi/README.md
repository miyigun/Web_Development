# surgu-ile-sekil-gosterimi

This repository contains an **interactive, browser-based educational activity** titled  
**“Sürgü ile Şekil Gösterimi”**.

The project is implemented as a **single-page web application (SPA-like)**.  
Learners use a **slider** to visualize the **four consecutive powers** of a non-zero real number **x** through **0D/1D/2D/3D physical models** (point, line segment, square, cube).  
The app includes a **hint dialog overlay**, **sound effects**, and equation visuals (embedded as **base64 image data**).

---

## Features

### 1. Slider-based Exploration (0 → 3)
- A range input (`#slider`) lets the learner switch between **four models**:
  - `0` → **0D**: point
  - `1` → **1D**: line segment with endpoints
  - `2` → **2D**: square with labeled vertices
  - `3` → **3D**: cube image
- The selected value is shown in `#sliderValue`
- The dimension label is shown in `#dimension`

### 2. Dynamic Shape Rendering
- The drawing area is the `#shape` container
- Shapes are built dynamically in `main.js` using DOM elements:
  - `.point`, `.line`, `.square`, `.cube`
  - `.label` for point/edge annotations (A, B, C, D, “x birim”)

### 3. Equation Display (Image)
- `#equation` is updated each time the slider changes
- Equation visuals are generated by `dataOfEquation(index)` and inserted as an `<img>`
- The cube image is provided by `kupVerisi()` (base64 image data)

### 4. Hint Dialog (Overlay)
- A hint dialog is shown on load via `ipucuDiyalogAc(baslik, ipucu)`
- Overlay styling is defined in `style.css`:
  - `.dialog-ipucu-overlay`, `.dialog-ipucu-box`, `.close-ipucu-button`
- The dialog uses an icon image: `images/key.png`

### 5. Audio Feedback
Preloaded audio elements in `index.html`:
- `mp3/click.mp3` (used on dialog close)
- `mp3/menu_option.mp3` (used when slider changes)

> Note: Browsers may block audio until the user interacts with the page.

### 6. Simple Responsive-Friendly Layout
- `html, body` uses flexbox centering
- Minimum width is limited (`min-width: 250px`)
- The main UI is contained in `#container`

---

## Project Structure

- `index.html` — main page UI (title, slider, shape area, equation area, audio tags)
- `style.css` — layout + shape styles + hint dialog overlay styles
- `main.js` — core logic (slider handler, shape generation, equations, sounds, hint dialog)
- `mp3/` — audio files
- `fonts/` — custom font (`fonts/Helveticasinav.ttf`)
- `icons/` — favicon (`icons/favicon.ico`)
- `images/` — `images/key.png` (hint dialog icon)

## Getting Started

### Run locally
1. Clone the repository:
   ```bash
   git clone https://github.com/miyigun/egik_prizma.git
   ```

2. Enter the folder:
   ```bash
   cd egik_prizma
   ```

3. Start a local server (recommended):
   ```bash
   python -m http.server 8000
   ```

4. Open in your browser:
   - `http://localhost:8000`

> Tip: Running via a local server is recommended for reliable loading of assets (fonts/audio/images).

## Notes
- Right-click is disabled on the page (`oncontextmenu="return false;"`). Remove it if you need browser context menu tools for debugging.
- The app uses a custom font via `@font-face`; make sure `fonts/Helveticasinav.ttf` exists.
- Equations and the cube image are embedded as base64 strings inside `main.js`, so no extra image files are required for them (except `images/key.png`).
