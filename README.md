# Web_Development

This repository contains **web development examples, learning experiments, and mini projects**.  
The goal is to learn and share modern, practical, and scalable approaches to building web applications.

---

## Folders / Categories

### 1. Experiments-Simple_Web
- Small experiments and quick prototypes  
- HTML / CSS / JavaScript practice projects  
- Simple UI/UX trials and component ideas  

### 2. Games-Simulation
- Browser-based mini games  
- Simulation and interactive logic examples  
- Projects focused on animation, physics-like behaviors, and user interaction  

### 3. Math-Educational_Tools
- Math-related tools and calculators  
- Educational mini apps and visualizations  
- Examples designed to support learning and practice  

---

## Getting Started

To run the projects:

1. Clone the repository:
   ```bash
   git clone https://github.com/miyigun/Web_Development.git
   ```

2. Navigate to the repository folder:
   ```bash
   cd Web_Development
   ```

3. Choose a subfolder and open the project:
   - If it’s a static web project, you can open `index.html` directly in the browser.
   - Or use a local server (recommended).

   Example (VS Code Live Server):
   - Open the folder in VS Code
   - Right click `index.html` → **Open with Live Server**
